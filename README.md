# LIVETV

Benvenuti nel repository LIVETV. Qui troverete diverse liste M3U per canali TV e sportivi.

## Requisiti Fondamentali

> [!IMPORTANT]
> Per il corretto funzionamento di quasi tutte le liste presenti in questo repository, è necessario l'utilizzo di **[EasyProxy](https://github.com/realbestia1/easyproxy)**.

### Liste che richiedono EasyProxy:
- `dlhd.m3u`
- `eventi_dlhd.m3u`
- `sports99.m3u`
- `sportsonline.m3u`
- `streamed.m3u`
- `vavoo.m3u`
- `world.m3u`
